Landing Inmobiliaria de Captación

Landing page enfocada en la generación de leads para bienes raíces, diseñada para convertir visitantes en clientes potenciales mediante una experiencia clara, moderna y orientada a resultados.


Objetivo del proyecto

Este proyecto fue desarrollado como una solución de captación de clientes para el sector inmobiliario.

El enfoque principal es:
- Maximizar la conversión
- Simplificar la experiencia del usuario
- Generar confianza desde el primer contacto

 Tecnologías utilizadas

- React
- Vite
- JavaScript
- CSS
- Responsive Design

Comportamiento del formulario

- Validación de campos obligatorios
- Simulación de envío (sin backend)
- Mensaje de confirmación:

> “Muchísimas gracias. Uno de nuestros asesores se contactará contigo muy pronto.”
